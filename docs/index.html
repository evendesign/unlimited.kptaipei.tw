文章類別查詢

用來取得目前官網文章類別清單

GET http://api.kptaipei.tw/v1/category/?accessToken=[YOUR_API_KEY]

[YOUR_API_KEY] 的部分在首頁可填寫申請 API KEY 。

資料範例：

{
  "isSuccess": true,
  "errorCode": 0,
  "data": [
    {
      "name": "【柯P新政】",
      "id": "40"
    },
    {
      "name": "【真・柯文哲】",
      "id": "41"
    },
    {
      "name": "新聞稿",
      "id": "42"
    }
  ],
  "errorMessage": null
}

類別內容查詢

GET http://api.kptaipei.tw/v1/category/<CATEGORY_ID>?accessToken=[YOUR_API_KEY]

相簿清單查詢

柯文哲每天的行程都有許多紀錄，在這你可以找到所有相簿

GET http://api.kptaipei.tw/v1/albums/?accessToken=[YOUR_API_KEY]

資料範例：

{
  "isSuccess": true,
  "errorCode": 0,
  "errorMessage": null,
  "data": [
    {
      "id": "72157646239459867",
      "photos": "15",
      "videos": 0,
      "title": "2014.8.20電信工程公會",
      "description": "",
      "date_create": 1408525510000,
      "date_update": 1408525530000,
      "link": "https://www.flickr.com/photos/110839995@N06/sets/72157646239459867/"
    }
  ],
  "pageInfo": {
    "totalResults": 1,
    "resultsPerPage": 500
  }
}

參數

GET page=?
第幾頁 (optional,default:0)
GET page_size=?
一次顯示幾筆 (optional,default:100,max:500)

相簿照片查詢

    "totalResults": 1,
    "resultsPerPage": 500

}

說明

small_square
75x75 正方形裁切
large_square
150x150 正方形裁切
thumbnail
裁切比較長的一邊為 100px
small
裁切比較長的一邊為 320px
medium
裁切比較長的一邊為 640px
large
裁切比較長的一邊為 1024px
original
原圖大小

參數

GET page=?
第幾頁 (optional,default:0)
GET page_size=?
一次顯示幾筆 (optional,default:100,max:500)
