"類別清單 /category "

可以用來取得目前官網文章類別清單

關於 API Key

在首頁可填寫申請 API KEY 。 

GET http://api.kptaipei.tw/v1/category/?accessToken=[YOUR_API_KEY]

"指定類別內的文章清單 /category/[CATEGORY_ID]"

給定類別 ID (可由 /category 取得) 後，可取得該類別底下文章清單。

參數

GET page=? 第幾頁(optional,default:0)

GET page_size=? 一次顯示幾筆(optional,default:100) 

GET http://api.kptaipei.tw/v1/category/?accessToken=[YOUR_API_KEY]
