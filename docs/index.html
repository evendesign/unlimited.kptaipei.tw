文章類別查詢

用來取得目前官網文章類別清單

GET http://api.kptaipei.tw/v1/category/?accessToken=[YOUR_API_KEY]

[YOUR_API_KEY] 的部分在首頁可填寫申請 API KEY 。

資料範例：

{
  "isSuccess": true,
  "errorCode": 0,
  "data": [
    {
      "name": "【柯P新政】",
      "id": "40"
    },
    {
      "name": "【真・柯文哲】",
      "id": "41"
    },
    {
      "name": "新聞稿",
      "id": "42"
    }
  ],
  "errorMessage": null
}
